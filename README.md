# Graph-Data-Structures-Implementation

Semestrální projekt z předmětu Datové struktury a algoritmy zaměřený na modelování dopravní sítě pomocí neorientovaného grafu.

Aplikace umožňuje pracovat se sítí obcí a komunikací, upravovat ohodnocení hran a vyhledávat nejkratší i alternativní trasy mezi zvolenými uzly. 
Součástí řešení je také práce s potenciálně problematickými komunikacemi, které mohou ovlivnit výsledný výpočet tras.

Projekt slouží jako demonstrační softwarová aplikace pro:
- reprezentaci dopravní infrastruktury pomocí grafu,
- vyhledávání uzlů a hran,
- úpravu a správu hran v síti,
- výpočet tras podle zadaných podmínek,
- otestování funkcionality prostřednictvím jednoduchého grafického rozhraní.

## Použité principy
- neorientovaný ohodnocený graf,
- práce s uzly a hranami,
- dynamický výpočet alternativních tras,
- abstraktní datové struktury,
- základní GUI pro vstupy a výstupy.

## Cíl projektu
Cílem projektu je ukázat praktické využití grafových struktur a algoritmů při řešení problému plánování tras v dopravní síti.
