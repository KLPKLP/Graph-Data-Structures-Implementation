# Graph-Data-Structures-Implementation

Semestrální projekt z předmětu Datové struktury a algoritmy vytvořený v jazyce C#.

Projekt je zaměřen na modelování dopravní sítě pomocí neorientovaného ohodnoceného grafu a na výpočet nejkratší i alternativních tras mezi vybranými uzly. Aplikace umožňuje pracovat s uzly a hranami grafu, upravovat ohodnocení komunikací a zohledňovat problémové úseky, které mohou ovlivnit výslednou trasu.

Součástí řešení je také jednoduché grafické uživatelské rozhraní pro testování jednotlivých funkcí aplikace.

## Funkcionalita
- reprezentace dopravní sítě pomocí grafu,
- vyhledávání a správa uzlů a hran,
- úprava ohodnocení hran,
- výpočet nejkratší trasy,
- generování alternativních tras,
- zohlednění potenciálně neprůjezdných komunikací.

## Použité technologie
- C#
- .NET
- grafické uživatelské rozhraní ve Windows Forms

## Cíl projektu
Cílem projektu je ukázat praktické využití grafových datových struktur a algoritmů při řešení úlohy plánování tras v dopravní síti.
